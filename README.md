# business-dashboard-ui

A business dashboard showing sales, revenue, and other key metrics Using this UI as seen on image. 
Users create workspace and within that workspace have a set of entities to share with other users.

## Setup

```sh
git clone https://github.com/GPT-Engineer-App/business-dashboard-ui.git
cd business-dashboard-ui
npm i
```

```sh
npm run dev
```

This will run a dev server with auto reloading and an instant preview.
